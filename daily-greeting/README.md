# Daily Greeting Plugin

A simple plugin that responds to greetings with today's date, a fun fact, and a motivational line.

## Purpose

This plugin exists to test cross-device plugin sync via a marketplace. The fun fact is versioned, so you can easily tell whether an update has propagated by checking which fact appears.

## Components

- **Skill: daily-greeting** - Triggers on casual greetings like "good morning" or "hello"
- **Command: /greet** - Manually trigger the greeting on demand
